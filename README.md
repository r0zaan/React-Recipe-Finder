A **ReactJS** application to search, view, and listen to recipes using the **TheMealDB API**.  
Includes features like **dark/light mode**, **text-to-speech**, and an **interactive recipe modal**.

[ðŸ”— Live Demo](https://react-recipe-finder.netlify.app/)

---

## Features

- **Search Recipes**: Search by recipe name using the TheMealDB API.
- **Recipe Modal**: View recipe details including ingredients, instructions, and category.
- **Text-to-Speech**: Play and stop recipe instructions using **Web Speech API**.
- **Dark & Light Mode**: Toggle between dark and light themes.
- **Interactive UI**: Hover effects, responsive design, and icon-enhanced interface.
- **YouTube Integration**: Watch recipe tutorial videos if available.
